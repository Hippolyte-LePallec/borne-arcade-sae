# 🎮 JavaSpace

## 📖 Description

JavaSpace est un jeu de tir spatial rétro, inspiré des classiques arcade. Vous incarnez un vaisseau spatial qui doit survivre aux vagues d'ennemis ennemies, collecter des bonus et affronter un boss redoutable.  L'univers du jeu est simple mais efficace : pixels colorés, bruitages vintage et une musique entraînante vous plongeront dans l'atmosphère intense des jeux arcade de la fin des années 80.

## 🎯 Objectif

Votre mission est simple mais périlleuse : survivre ! Évitez les tirs ennemis, détruisissez les vagues d'ennemis qui s'acharnent sur votre vaisseau et récoltez les bonus pour améliorer vos capacités. En fin de partie, affrontez le boss redoutable et mettez un terme à son règne de terreur.

## 🕹️ Comment jouer

* **Contrôles:** Utilisez les touches directionnelles du clavier (flèches haut, bas, gauche, droite) pour déplacer votre vaisseau dans l'espace.
* **Tir:** Appuyez sur la touche "Espace" pour tirer des lasers.


* **Bonus:**  Des bonus apparaissent aléatoirement sur le terrain de jeu. Collectez-les pour améliorer vos capacités (plus de vie, tirs plus rapides, etc.).

* **Ennemis:** Des vagues d'ennemis attaquent régulièrement votre vaisseau. Utilisez vos tirs pour les détruire avant qu'ils ne vous atteignent.


## ⌨️ Contrôles

| Touche | Action |
|---|---|
| Flèche Haut | Déplacer le joueur vers le haut |
| Flèche Bas | Déplacer le joueur vers le bas |
| Flèche Gauche | Déplacer le joueur vers la gauche |
| Flèche Droite | Déplacer le joueur vers la droite |
| Espace | Tirer |

## 💡 Conseils

* Soyez agile et réactif pour éviter les tirs ennemis.
* Collectez les bonus pour augmenter vos chances de survie.
* Apprenez les mouvements des ennemis pour anticiper leurs attaques.
* N'oubliez pas que chaque niveau devient de plus en plus difficile !
