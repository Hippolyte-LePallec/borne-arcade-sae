# 🎮 DinoRail

## 📖 Description

DinoRail est un jeu d'arcade rétro où vous contrôlez un dinosaure agile qui doit éviter les obstacles en sautant sur un rail.  🚀🦖

L'ambiance est dynamique et trépidante, avec des graphismes simples et colorés rappelant les jeux arcade classiques. Votre mission est simple : parcourir le maximum de distance possible sans toucher aux cactus !🌵💥

## 🎯 Objectif

Le but du jeu est de rester en vie le plus longtemps possible en évitant les obstacles (des cactus).  Chaque cactus que vous évitez vous rapporte des points.  🏆

Le jeu se termine lorsque vous heurtez un cactus.

## 🕹️ Comment jouer

Pour progresser dans DinoRail, vous devez contrôler votre dinosaure :

* **Déplacement :** Le dinosaure se déplace automatiquement vers la droite sur le rail.
* **Saut :** Appuyez sur une touche pour faire sauter votre dinosaure et éviter les cactus. 

## ⌨️ Contrôles

| Touche | Action |
|---|---|
|  Espace | Sauter |


## 📜 Règles

* Le jeu se termine lorsque le dinosaure touche un cactus.  💀
* Chaque cactus évité rapporte des points.
*  Plus vous progressez, plus les cactus apparaissent fréquemment. ⚡️

## 💡 Conseils

* **Pratiquez votre timing !** L'art du saut est essentiel pour éviter les cactus.
* **Soyez attentif !** Observez attentivement l'arrivée des cactus et préparez vos sauts.
* **N'abandonnez pas !**  Même si vous échouez, continuez à jouer et améliorez votre score !

Bonne chance dans DinoRail! 🦖🏆
