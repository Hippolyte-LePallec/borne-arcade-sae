## 🎮 TronGame


### 📖 Description

TronGame est un jeu vidéo d'arcade inspiré du classique Tron. Deux joueurs contrôlent des vaisseaux lumineux qui se battent sur une grille en s'affrontant et en créant des traînées de lumière. Le but du jeu est de piéger l'adversaire dans son propre circuit ou de le faire dépasser la limite du plateau.  

### 🎯 Objectif

Le but du jeu est simple : survivre plus longtemps que votre adversaire ! Vous pouvez accomplir cet objectif en:

* **Piégeant votre adversaire** : créez un labyrinthe complexe qui l'enferme dans son propre circuit, le forçant à traverser ses traînées et perdre.
* **Évitez les traînées de lumière**  : les traînées que vous créez sont des obstacles pour vous et votre adversaire. Ne vous laissez pas piéger par vos propres créations !
* **S'échapper du périmètre** : si vous sortez du plateau, vous êtes éliminé. Gardez une vigilance accrue aux limites !

### 🕹️ Comment jouer

Le jeu se déroule sur une grille divisée en cellules. Vous contrôlez votre vaisseau en utilisant les touches directionnelles pour le déplacer.  Votre vaisseau laisse une traînée de lumière derrière lui à chaque déplacement. Si vous colisionnez avec la traînée d'un autre joueur ou si vous quittez le plateau, vous perdez.

### ⌨️ Contrôles

| Joueur | Action | Touche |
|---|---|---|
| **Joueur 1** |  Haut | ↑ |
|  | Bas | ↓ |
|  | Gauche | ← |
|  | Droite | → |
| **Joueur 2** | Haut | O |
|  | Bas | l |
|  | Gauche | k |
|  | Droite | m |

### 📜 Règles

* Le premier joueur à piéger l'autre ou à le faire dépasser le plateau gagne la partie.
* Une fois qu'un joueur touche sa propre traînée, il est éliminé.
* Les joueurs peuvent se déplacer sur toute la grille à condition de ne pas sortir des limites.


### 💡 Conseils

* **Anticipations :** Essayez de prévoir les mouvements de votre adversaire pour créer un piège efficace.
* **Blocage stratégique :** N'hésitez pas à bloquer le passage de votre adversaire en construisant des traînées de lumière complexes.
* **Exploration :** Explorez la grille et essayez différentes stratégies pour trouver la meilleure façon de piéger votre adversaire.
