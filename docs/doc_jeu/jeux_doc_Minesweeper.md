# 🎮 Minesweeper

## 📖 Description
Minesweeper est un jeu classique de logique et de stratégie. Votre objectif est de déterrer toutes les cases du plateau sans toucher aux mines cachées. Chaque case découverte révèle un nombre indiquant le nombre de mines présentes dans les cases adjacentes. Utilisez ces indices pour raisonner et dévoiler le plateau en toute sécurité !

## 🎯 Objectif
Le but du jeu est de découvrir toutes les cases vides du plateau sans faire exploser aucune mine. Vous gagnez si vous réussissez à déterrer toutes les cases sauf celles contenant des mines. Attention, toucher une mine provoque la fin du jeu !

## 🕹️ Comment jouer
Le jeu se déroule sur un plateau rectangulaire divisé en cases. Chaque case peut contenir :
* **Une mine:**  Représentée par un symbole spécifique (voir "Règles"). Vous devez éviter de cliquer dessus.
* **Un nombre:** Ce nombre indique le nombre de mines présentes dans les cases adjacentes à cette case.

En cliquant sur une case, vous la découvrirez :

* Si la case contient une mine, le jeu est terminé et vous perdez.
* Si la case ne contient pas de mine, le nombre affiché (si il y en a un) vous indique combien de mines sont autour de cette case. 
* Vous pouvez cliquer sur les cases à proximité pour découvrir davantage d'indices et démêler l'emplacement des mines.

## ⌨️ Contrôles
Pour jouer, utilisez votre clavier ou la souris :

| Action | Touche/Bouton  |
|---|---|
| Découvrir une case | **Clic gauche** (souris) / Entrer (clavier) |
| Poser un drapeau sur une case suspectée de contenir une mine | Clic droit (souris) / R (clavier) |
| Quitter le jeu | Echap (clavier) |

## 📜 Règles
* **Conditions de victoire:** Découvrir toutes les cases vides sans toucher aux mines.
* **Conditions de défaite:** Toucher une mine.
* **Découvre d'une case:**  Une case découverte montre soit un nombre, indiquant le nombre de mines dans les cases adjacentes (8 cases autour), soit rien si elle ne contient pas de mine. 

## 💡 Conseils
* Commencez par cliquer sur les coins du plateau. Ils sont souvent moins risqués car ils ont moins de cases adjacentes.
* Notez attentivement les nombres que vous voyez. Ils sont des indices précieux pour identifier l'emplacement des mines.
* Soyez patient et réfléchissez à chaque mouvement !
