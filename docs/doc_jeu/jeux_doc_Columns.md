# 🎮 Columns

## 📖 Description

Columns est un jeu de puzzle classique où l'objectif est de faire disparaître des lignes entières de gemmes en les faisant se chevaucher. Vous contrôlez une colonne de gemmes qui descend lentement, et vous devez faire pivoter les gemmes pour aligner des groupes de trois ou plus du même type. 

Le jeu offre un défi stimulant et addictif, avec des graphismes colorés et un gameplay fluide. Testez vos capacités logiques et vos réflexes !

## 🎯 Objectif

L'objectif du jeu est d'éliminer le plus grand nombre possible de lignes de gemmes en les alignant par groupes de trois ou plus. Pour ce faire, vous devez faire pivoter les gemmes dans la colonne afin de les aligner horizontalement ou verticalement.

Les lignes qui disparaissent libèrent de l'espace et les gemmes supérieures descendent pour remplir le vide. 

Le jeu se termine lorsque les gemmes atteignent la ligne du haut de l'écran.

## 🕹️ Comment jouer

- **Contrôles:** Utilisez les touches directionnelles ou les boutons du joystick pour faire pivoter la colonne de gemmes.
- **Objectif:** Alignez trois ou plus de gemmes du même type horizontalement ou verticalement pour les faire disparaître.
- **Score:** Vous gagnez des points en fonction du nombre de lignes éliminées à la fois.

## ⌨️ Contrôles

| Action | Touche/Bouton |
|---|---|
| Déplacement vers le haut | ↑ |
| Déplacement vers le bas | ↓ |
| Déplacement vers la gauche | ← |
| Déplacement vers la droite | → |
| Rotation de la colonne | A / X |

## 📜 Règles

- Les gemmes ne peuvent être déplacées que verticalement et horizontalement.
- Vous ne pouvez pas déplacer les gemmes à l'extérieur du cadre du jeu.
- Le jeu se termine lorsque les gemmes atteignent la ligne du haut de l'écran.
- Il n'y a pas de limite au nombre de lignes que vous pouvez éliminer en une seule fois.

## 💡 Conseils

- Essayez d'aligner des groupes de gemmes aussi grands que possible pour obtenir plus de points.
- Planifiez vos déplacements à l'avance pour maximiser les combinaisons possibles.
- Soyez attentif aux gemmes qui descendent et essayez de les bloquer pour créer des lignes stables.
