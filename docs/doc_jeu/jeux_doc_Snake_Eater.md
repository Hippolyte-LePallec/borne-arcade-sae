# 🎮 Snake_Eater

## 📖 Description
Snake_Eater est un jeu d'arcade rétro où vous incarnez un serpent qui grandit en mangeant des pommes. Votre but est de survivre le plus longtemps possible tout en augmentant votre score.  Le jeu se déroule sur un plateau limité, et chaque collision avec les murs ou votre propre corps vous fait perdre la partie.

## 🎯 Objectif
Votre objectif dans Snake_Eater est simple : manger le maximum de pommes possibles sans entrer en collision avec les bords du plateau ou avec votre propre queue. Plus vous mangez de pommes, plus votre serpent grandit et plus il devient difficile de survivre !

## 🕹️ Comment jouer
Utilisez les touches fléchées pour déplacer votre serpent dans les quatre directions (haut, bas, gauche, droite).  Votre serpent suivra toujours la direction indiquée par le dernier mouvement. 

## ⌨️ Contrôles
* **Flèche du Haut:** Déplacez votre serpent vers le haut
* **Flèche du Bas:** Déplacez votre serpent vers le bas
* **Flèche de Gauche:** Déplacez votre serpent vers la gauche
* **Flèche de Droite:** Déplacez votre serpent vers la droite

##  🔥 Astuces
* Essayez de prévoir les déplacements de votre serpent pour éviter les collisions.
* Changez de direction fréquemment pour éviter d'être coincé dans un coin.
* Augmentez progressivement la vitesse de jeu lorsque vous devenez plus confiant.
